Joomla

There are 2 ways to connect Funnelll with Joomla Launch Website. We will cover both in this tutorial.

To get started, login or create an account on Funnelll.com

After logging in, input your website's information

After inputting your website's information, the following page is where you connected your site with Funnelll.

First method: Adding Funnelll Code into your Joomla Site

You will see installation information and some code that needs to be added to your Joomla site. Leave this page open, you will need to paste this code later.

To do this, head over to your Joomla Launch control panel. Click the Templates section on the Control Panel menu.

Here you will see a gold star next to the name of Style is your default (in this case, that would be protostar), this is the template that will need to be edited in the Templates section. Click on Templates from the menu on the left.

Look for your active template on this page and click on it.

You will find various files of the template, locate the index.php file.

Scroll down in the index.php file until you spot the <head> tag in the code. This is were you will add the Funnelll code that you received during the Funnelll setup.

Second Method: Installing Google Tag Manager Extension to your Joomla Site

Locate the Extensions section on your Joomla Launch Control Panel.

You will search for "Google Tag Manager" under the "Install from Web" tab.

Install the following extension.

After a successful installation message, you need to Enable the extension in the Manage section. To do that, click on Manage from the left menu.

Search for the Google Tag Manager and click on the red cross next to its name to enable it.

After is has been enabled, you need to input your Google Tag Manager Container ID. To do that, locate the Plugins section from the Extensions tab from the top navigation bar.

Click on the Google Tag Manager to configure it.

Add the Container ID (can find the container ID on google tag manager account) and hit save, you can now head back to the Funnelll and click Verify to check if installation is successful.
